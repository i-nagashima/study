/* JSXでの{}
------------------------------------------------------------ */
JSXでは変数や関数は{}の中に書く
そのため、アロー関数、無名関数の{}は変数や関数として展開されてしまうので
{} → () もしくは 書かない
でやる
Array.map((value) => {

})
↓
Array.map((value) => (

))
or
Array.map((value) =>

)

/* JSXでのclass (css)
------------------------------------------------------------ */
classはだめ。JavaScriptのclassと区別つかないから
classNameとする

/* JSXでのイベント
------------------------------------------------------------ */
onclickはだめ。
onClickとする。他のイベントも同じ

/* JSXでの終了タグがないタグ
------------------------------------------------------------ */
<input>はだめ。
<input />とする。
